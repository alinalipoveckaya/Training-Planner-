# Training Planner

**Автор:** [Липовецкая Алина Сергеевна]

**дата:**2026.04.30
## Описание

Приложение для планирования тренировок с графическим интерфейсом. Позволяет:
* Добавлять тренировки с указанием даты, типа и длительности.
* Фильтровать записи по типу тренировки и периоду.
* Сохранять и загружать данные в формате JSON.

## Как использовать

1. Запустите приложение: `python main.py`.
2. Заполните поля:
   * **Дата** — укажите в формате ГГГГ-ММ-ДД.
   * **Тип тренировки** — выберите из списка или введите свой.
   * **Длительность** — введите положительное число (в минутах).
3. Нажмите **«Добавить тренировку»** для сохранения записи.
4. Используйте фильтры для отбора данных:
   * Выберите тип тренировки из выпадающего списка.
   * Укажите период в полях «С даты» и «По дату».
5. Нажмите **«Применить фильтр»** для обновления таблицы.
6. Нажмите **«Сбросить фильтр»**, чтобы отобразить все тренировки.

## Примеры использования

**Тест 1: Добавление тренировки**
1. Введите: Дата = 2024-01-15, Тип = Кардио, Длительность = 60.
2. Нажмите «Добавить тренировку».
3. Убедитесь, что запись появилась в таблице.

**Тест 2: Фильтрация**
1. Установите фильтр: Тип = Йога, С даты = 2024-01-01, По дату = 2024-01-31.
2. Нажмите «Применить фильтр».
3. Проверьте, что отображаются только тренировки типа «Йога» за январь.

**Тест 3: Сброс фильтров**
1. Нажмите «Сбросить фильтр».
2. Убедитесь, что таблица обновилась и показывает все записи.
